
> Problem: [231. 2 的幂](https://leetcode.cn/problems/power-of-two/description/)

[TOC]

# 思路

> $2$ 的幂次方的数的二进制特征。

# 解题过程

> 先利用 `n&(-n)` 可求出某二进制数的最后一个二进制 $1$ 的位置，若该位置恰好为最高位，证明为 $2$ 的幂次方。

# 复杂度

- 时间复杂度: $O(1)$
- 空间复杂度: $O(1)$



# Code
```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        return n > 0 && (n & -n) == n;
    }
};

```
