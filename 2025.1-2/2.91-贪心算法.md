#### [860. 柠檬水找零](https://leetcode.cn/problems/lemonade-change)

**分析**：本文为贪心算法，每道题最后会随缘给出证明。

```cpp
class Solution {
public:
    bool lemonadeChange(vector<int>& bills) {
        int five = 0, ten = 0;
        for (auto x : bills) {
            if (x == 5) five++;
            else if (x == 10) {
                if (five == 0) return false;
                five--; ten++;
            } else {
                if (ten && five) {
                    ten--, five--;
                } else if (five >= 3) {
                    five -= 3;
                } else {
                    return false;
                }
            }
        }
        return true;
    }
};
```

**证明**：略。
